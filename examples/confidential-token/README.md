## What this proves

- **EXAMPLE-01:** The `zama-skills` plugin produces real, deployable, production-grade output — every file in `packages/contracts/` and `packages/frontend/` was scaffolded by `/zama-skills:init`, `/zama-skills:contract`, `/zama-skills:test`, `/zama-skills:deploy`, and `/zama-skills:frontend` against an empty directory.
- **EXAMPLE-03:** The fhEVM relayer + ACL flow round-trips through a real wallet on Sepolia — connect MetaMask, mint, decrypt your balance via the relayer, and transfer encrypted amounts.
- **No hallucinated APIs:** Every dependency is pinned from `shared/pinned-versions.json` and verified live via context7 (`/zama-ai/fhevm`, 1,772 snippets) at scaffold time.

## How this was built

This entire example was generated by running the `zama-skills` plugin's slash commands in sequence:

1. `/zama-skills:init confidential-token` — pnpm monorepo scaffold (Hardhat + Next.js)
2. `/zama-skills:contract` — `ERC7984` base + `mint` + ACL `FHE.allowThis` discipline
3. `/zama-skills:test` — fhEVM mock + Sepolia integration tests
4. `/zama-skills:deploy --sepolia` — deploy + Etherscan verification + Registry check
5. `/zama-skills:frontend` — `@zama-fhe/relayer-sdk` + `useDecrypted` 4-state UX + `EncryptedInput`
6. Hand-curated polish: Next.js 15 + shadcn/ui

See [`.gsd-snapshot.json`](./.gsd-snapshot.json) for the exact skill commit SHAs used to scaffold this example — every file is reproducible by replaying the same skill versions.

## Local dev

```bash
pnpm install                                                # install workspace deps
pnpm --filter contracts hardhat compile                     # compile contracts
cp packages/frontend/.env.local.example packages/frontend/.env.local  # then fill keys
pnpm --filter frontend dev                                  # http://localhost:3000
```

Required env vars in `packages/frontend/.env.local`:

| Variable | Default / Where to get it |
| --- | --- |
| `NEXT_PUBLIC_CONTRACT_ADDRESS` | `0x04Bd105DE7a5D3297c3747cef90ac8b760136896` (already pre-filled) |
| `NEXT_PUBLIC_SEPOLIA_RPC` | Alchemy / Infura / `https://ethereum-sepolia.publicnode.com` |
| `NEXT_PUBLIC_RELAYER_URL` | `https://relayer.testnet.zama.org` (already pre-filled) |
| `NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID` | [cloud.walletconnect.com](https://cloud.walletconnect.com) |
